# 🏨 Hotel Booking Analysis: Revenue & Cancellation Intelligence 📊

> **"Turning 119K+ raw booking records into actionable financial growth strategies."**

---

### 📋 Project Overview
This project goes beyond basic visualization. I performed an end-to-end **Exploratory Data Analysis (EDA)** and **Unsupervised Machine Learning** to optimize hotel operations, identify high-risk cancellations, and segment customers for targeted marketing.

### 🛠️ The Technical Stack
* **Analytics:** Python (Pandas, NumPy)
* **Visualization:** Matplotlib, Seaborn
* **Advanced ML:** Dimensionality Reduction (**PCA**), Clustering (**K-Means**)
* **Environment:** Jupyter Notebook / Google Colab

---

### 🔍 Key Business Insights
* **Cancellation Risk:** High lead times are the #1 predictor of churn. 📉
* **Revenue Drivers:** Transient guests in City Hotels generate the highest ADR (Average Daily Rate). 💰
* **Market Trends:** August is the peak "Gold Mine" for bookings; Portugal & UK are the primary markets. 🌍
* **Stay Patterns:** 82.4% of guests are short-term stayers (1-4 nights).

---

### 🧠 Advanced Methodology
Unlike standard EDA, this project incorporates:
1.  **PCA (Principal Component Analysis):** Reducing feature noise while maintaining variance.
2.  **K-Means Clustering:** Segmenting guests into distinct personas (Alpha, Delta, Charlie) to enable personalized revenue management.
